# BountyDash
## 프로젝트 소개
- 예제로 배우는 언리얼 엔진 4 게임 개발 5장 ~ 6장 진행 및 진행 과정 기록 레포지토리
### 게임 분류
- 종스크롤 러닝 게임
### 사용 엔진 및 언어
- 언리얼엔진
- C++
- 블루프린트의 사용을 최소로 하고 대부분의 모든 기능을 C++로 구현
### 게임 진행 방식
>A, D => 좌, 우 이동   
>Space => 점프
